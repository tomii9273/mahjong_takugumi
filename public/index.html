麻雀大会用卓組表

麻雀など、多人数で行うゲーム・スポーツの大会やイベント等に使用できる卓組表（対戦表）、およびその作成方法をまとめました。

  • 直接卓組表にリンクされているものに関しては、当サイトの作者が作成したものです。二次配布や改変を含め、ご自由にお使いください。
    • 各卓組は縦軸がプレイヤー、横軸が対戦、番号が卓番号を指します（プレイヤー番号・対戦番号は振ってありません）。
    • 卓組はすべてタブ区切りのtxt形式です。全選択しコピーペーストすることでエクセルにも（表として）貼り付けられます。
    • 卓組の作成方法・使用したソースコードを公開しています。まだ一部ですが、今後増やしていく予定です。
  • 外部サイトにリンクされているものに関しては、そのサイトの使用規則に従ってください。
  • 誤字・ミス・不具合等がありましたら、X (@Tomii9273) までご連絡いただけると嬉しいです。
  • 当サイトのソースコード・更新履歴は GitHub で公開しています。

1 対 1 のゲーム用

1 対 1 で行うゲームやスポーツでは、以下のリンク先の方法で簡単に総当たり戦を組むことができます。
https://www.ne.jp/asahi/office/okada/soft/souatari/help/help.html
人数 (チーム数) が偶数の場合は「n 人が同時並行で n / 2 戦を行うこと」を n-1 回行うことになります。
奇数の場合は、1 多い偶数で総当たり戦を作成した後、選手番号 1 番を削除し、1 番と対戦する人を抜け番とする方法をとることで
「n-1 人が同時並行で (n-1) / 2 戦を行うこと」を n 回行うことになります (画像は 7 人の例です)。

また、麻雀総当たり戦の卓組作成方法 に記載した「円と点を使った方法」でも、以下のリンク先のようにして総当たり戦を組むことができます。

5 人以上で対戦するゲーム用

n 人で対戦するゲームについて、n が素数の場合には n 卓 n^2 人 n+1 戦の総当たり戦を組めます。
http://math.a.la9.jp/aleague4.htm
上記リンク先の下から 2 番目の解答 (カラフルな図があるもの) の方法で可能です。卓組表を作成するエクセルもあります。
ただし、(リンク先には書かれていませんが) この方法では n が素数でない場合には総当たり戦にならず、重複や未対戦が生じます。

この方法を一般化して記述してみました。
(各種番号を 1 番ではなく 0 番から振っていることに注意してください。)

  1. 選手に 0 ～ n^2 - 1 の番号を振る。
  2. 整数 p を n で割った商を a(p)、余りを b(p) とする。
  3. 0 ～ n-1 戦目では、選手 p の s 戦目の卓番号を「a(p) × s + b(p) を n で割った余り」とする。
  4. n 戦目 (最終戦) では、選手 p の卓番号を a(p) とする。
総当たり戦になることの証明

以下の 2 点を示せればよいです。

  1. 0 ≦ s ≦ n, 0 ≦ t ≦ n-1 のとき、s 戦目に卓番号 t である選手の数は、ちょうど n 人である。
  2. 0 ≦ p0 ≦ n^2-1, 0 ≦ p1 ≦ n^2-1, p0 ≠ p1 のとき、2 選手 p0, p1 が同卓する対戦番号 s (0 ≦ s ≦ n) はただ 1 つに定まる。

1 の証明

s = n のとき、選手番号が小さい方から n 人が第 0 卓、次に小さい n 人が第 1 卓、… につくため、各卓番号の選手数はちょうど n 人である。
以下、0 ≦ s ≦ n-1 で考える。
0 ≦ p ≦ n-1 のとき、選手 p の s 戦目の卓番号 t は t ≡ a(p) × s + b(p) = b(p) (mod n) を満たす。
各 s について、t は 0 から n-1 の値を 1 個ずつとる。
n ≦ p ≦ 2n-1 のとき、選手 p の s 戦目の卓番号 t は t ≡ a(p) × s + b(p) = s + b(p) (mod n) を満たす。
各 s について、s と n は互いに素なので、t は 0 から n-1 の値を 1 個ずつとる。
2n ≦ p ≦ 3n-1 のとき、選手 p の s 戦目の卓番号 t は t ≡ a(p) × s + b(p) = 2s + b(p) (mod n) を満たす。
各 s について、2s と n は互いに素なので、t は 0 から n-1 の値を 1 個ずつとる。
同様に考えると、0 ≦ p ≦ n^2-1 のとき、各 s について、t は 0 から n-1 の値を n 個ずつとる。
よって、s 戦目に卓番号 t である選手の数は、ちょうど n 人である。

2 の証明

2 選手 p0, p1 が同卓するとき、0 ≦ s ≦ n-1 ならば、a(p0) × s + b(p0) ≡ a(p1) × s + b(p1) (mod n) である。
変形して、(a(p0) - a(p1)) × s ≡ b(p1) - b(p0) (mod n) … (A)
a(p0) = a(p1) のとき、p0 ≠ p1 より b(p0) ≠ b(p1) なので (A) の解はないが、2 選手 p0, p1 は n 回戦で同卓するため、対戦番号 s はただ 1 つ (n) に定まる。
a(p0) ≠ a(p1) のとき、-n+1 ≦ a(p0) - a(p1) ≦ n-1 より、a(p0) - a(p1) と n は互いに素なので、s ≡ (b(p1) - b(p0)) × (a(p0) - a(p1))^(-1) (mod n) となり、s はただ 1 つに定まる。
よって、示された。
