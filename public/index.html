麻雀大会用卓組表

麻雀など、多人数で行うゲーム・スポーツの大会やイベント等に使用できる卓組表 (対戦表)、およびその作成方法をまとめました。

  • 外部サイトへ飛ばず、転載表記のない卓組表は、当サイトの作者が作成したものです。二次配布や改変を含め、ご自由にお使いください。
    • 多くの卓組表はそのまま印刷が可能です。また、全選択しコピー・ペーストすることでエクセルやスプレッドシートにも貼り付けられます。
    • 卓組の作成方法・使用したソースコードを公開しています。
  • 外部サイトへ飛ぶもの、または転載表記のあるものに関しては、元サイトの使用規則に従ってください。
  • 誤字・ミス・不具合等がありましたら、X (@Tomii9273) までご連絡いただけると嬉しいです。
  • 当サイトのソースコード・更新履歴は GitHub で公開しています。

4 人麻雀用

個人戦・4 人チーム戦用

  • 一般的な個人戦用の卓組です。
  • 第 1 戦の卓をチームとみなし、実際の対局は第 2 戦から始めることで、4 人 × (卓数) チームによるチーム戦を行えます。(重複同卓なしの卓組のみ)
  • 参加者数が 4 の倍数でない場合は、下の方にある「抜け番あり」の卓組を参照してください。
  • 卓組の作成方法はこちら: 重複同卓なし 重複同卓あり

6 卓以下

  • ◎ … 重複同卓なしの卓組です。どの 2 人組も複数回同卓することはありません。左から必要な対戦数分ご使用ください。
  • ○ … 重複同卓がありますが、なるべく少なく平等になるようにした卓組です。(完璧ではない可能性があります。詳細な条件はこちら)
  • 同じ卓数の卓組は、なるべく序盤戦部分が共通になるようにしています。
    これにより「3 卓で 4 戦やる予定だったが早く終わりそうなので、途中で 3 卓 5 戦の卓組に切り替える」といったことがしやすくなっています。

ゲストを呼ぶ大会用

  • (ゲスト以外の) 全選手がゲストと必ず同卓できる卓組です。
  • 各選手のゲストとの同卓回数がなるべく平等になるように、また同じ選手とばかり何度も同卓することがなるべく無いようになっています。(完璧ではない可能性があります。詳細な条件はこちら)

  • 上記条件を満たす卓組が存在しない場合は、表では「×」となっています。

4 チーム戦用

  • この卓組では、上から (卓数) 人ごとに区切って 4 つにチーム分け (例えば 10 卓 40 人の場合、上から 1 ～ 10 番目の選手をチーム A、11 ～ 20 番目をチーム B ...) したとき、
    各プレイヤーは自分と同じチームの選手とは当たらないため、4 チーム戦を行うことができます。
  • 重複同卓なしの卓組です。どの 2 人組も複数回同卓することはありません。左から必要な対戦数分ご使用ください。
  • 東南西北を平等にした個人戦の卓組を作りたい場合も、こちらの卓組を使いチームごとに座位を割り振ると (1 戦目は「A東, B南, C西, D北」、2 戦目は「A南, B西, C北, D東」など)、比較的平等になります。
  • 卓組の作成方法・ソースコードはこちら

ペア戦 (2 人チーム戦) 用

  • この卓組では、プレイヤーを上から 2 人ずつ区切ると、ペア戦 (2 人チーム戦) を行うことができます。
  • 重複同卓なしの卓組です。どの 2 人組も複数回同卓することはありません。左から必要な対戦数分ご使用ください。
  • ここにないペア数でペア戦を行う場合は、「個人戦・4 人チーム戦用」の卓組について、第 1 回戦の卓番号をペア決め用とみなし (同じ卓の 4 人で 2 ペアできる) 、第 2 回戦以降を実際の卓組として利用することをお勧めします。

総当たり戦用

(注...この項目の卓組のうち 28 卓以下のものは「個人戦・4 人チーム戦」にあるものとまったく同じです。)

  • 使用卓数が (3n+1) 卓 (参加人数が (12n+3) 人) のとき、(4n+1) 戦の重複のない総当たりを行うことができるとされています (現在、n ≦ 12 において確実)。全戦を行うと参加者全員とちょうど 1 回ずつ同卓することができます。
  • 総当たりをするほどたくさんの対局を行わない場合でも、前半の数回戦分を抜き出すことで、重複同卓なしの卓組として使用できます。

1 対 1 のゲーム用

1 対 1 で行うゲームやスポーツでは、以下のリンク先の方法で簡単に総当たり戦を組むことができます。
https://www.ne.jp/asahi/office/okada/soft/souatari/help/help.html
人数 (チーム数) が偶数の場合は「n 人が同時並行で n / 2 戦を行うこと」を n-1 回行うことになります。
奇数の場合は、1 多い偶数で総当たり戦を作成した後、選手番号 1 番を削除し、1 番と対戦する人を抜け番とする方法をとることで

5 人以上で対戦するゲーム用

n 人で対戦するゲームについて、n が素数の場合には以下の方法で n 卓 n^2 人 n+1 戦の総当たり戦を組めます。
http://math.a.la9.jp/aleague4.htm
上記リンク先の下から 2 番目の解答 (カラフルな図があるもの) の方法で可能です。卓組表を作成するエクセルもあります。
ただし、(リンク先には書かれていませんが) この方法では n が素数でない場合には総当たり戦にならず、重複や未対戦が生じます。

この方法を一般化して記述してみました。
(各種番号を 1 番ではなく 0 番から振っていることに注意してください。)

  1. 選手に 0 ～ n^2 - 1 の番号を振る。
  2. 整数 p を n で割った商を a(p)、余りを b(p) とする。
  3. 0 ～ n-1 戦目では、選手 p の s 戦目の卓番号を「a(p) × s + b(p) を n で割った余り」とする。
  4. n 戦目 (最終戦) では、選手 p の卓番号を a(p) とする。
総当たり戦になることの証明

以下の 2 点を示せればよいです。

  1. 0 ≦ s ≦ n, 0 ≦ t ≦ n-1 のとき、s 戦目に卓番号 t である選手の数は、ちょうど n 人である。
  2. 0 ≦ p0 ≦ n^2-1, 0 ≦ p1 ≦ n^2-1, p0 ≠ p1 のとき、2 選手 p0, p1 が同卓する対戦番号 s (0 ≦ s ≦ n) はただ 1 つに定まる。

1 の証明

s = n のとき、選手番号が小さい方から n 人が第 0 卓、次に小さい n 人が第 1 卓、… につくため、各卓番号の選手数はちょうど n 人である。
以下、0 ≦ s ≦ n-1 で考える。
0 ≦ p ≦ n-1 のとき、選手 p の s 戦目の卓番号 t は t ≡ a(p) × s + b(p) = b(p) (mod n) を満たす。
b(p) は 0 から n-1 の値を 1 個ずつとるので、各 s について、t は 0 から n-1 の値を 1 個ずつとる。
n ≦ p ≦ 2n-1 のとき、選手 p の s 戦目の卓番号 t は t ≡ a(p) × s + b(p) = s + b(p) (mod n) を満たす。
b(p) は 0 から n-1 の値を 1 個ずつとるので、各 s について、t は 0 から n-1 の値を 1 個ずつとる。
同様に考えると、0 ≦ p ≦ n^2-1 のとき、各 s について、t は 0 から n-1 の値を n 個ずつとる。
よって、s 戦目に卓番号 t である選手の数は、ちょうど n 人である。

2 の証明

2 選手 p0, p1 が同卓するとき、0 ≦ s ≦ n-1 ならば、a(p0) × s + b(p0) ≡ a(p1) × s + b(p1) (mod n) である。
変形して、(a(p0) - a(p1)) × s ≡ b(p1) - b(p0) (mod n) … (A)
a(p0) = a(p1) のとき、p0 ≠ p1 より b(p0) ≠ b(p1) なので (A) の解はないが、2 選手 p0, p1 は n 戦目で同卓するため、対戦番号 s はただ 1 つ (n) に定まる。
a(p0) ≠ a(p1) のとき、-n+1 ≦ a(p0) - a(p1) ≦ n-1 より、a(p0) - a(p1) と n は互いに素なので、s ≡ (b(p1) - b(p0)) × (a(p0) - a(p1))^(-1) (mod n) となり、s はただ 1 つに定まる。
よって、示された。
